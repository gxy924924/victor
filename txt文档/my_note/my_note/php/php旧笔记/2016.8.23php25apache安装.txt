==》虚拟目录
提一个需求：我的apache是安装在c盘，但是c盘没有空间，d盘有空间，能不能把目录放在d盘？


1配置虚拟目录在apache的conf目录下的httpd.conf
direcotory相当于欢迎页面
alias /mybliog "d:/myblog" 你的站点别名
order allow deny
allow from all访问权限设置

#配置虚拟目录
<IfModule dir_module>
#欢迎页面设置
    DirectoryIndex index.html
#你的站点别名
 Alias /mysite "D:/mysite"
<Directory d:/mysite>
#访问权限设置
Order allow,deny
Allow from all
	</Directory>
</IfModule>

2注销documentroot路径
# documentroot。。。。。。。

3测试
http://localhost/myblog/news.html

4如何设置欢迎页面
#欢迎页面设置
    DirectoryIndex index.html news.html hello.html .......

5关于访问权限
<directory >
order allow,deny  order顺序 allow允许 deny 表示拒绝 表示先看许可（allow）后看拒绝（deny）
allow from all    先看看（allow）后看拒绝（deny）  allow from all许可所有IP
</directory>
